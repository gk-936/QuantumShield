Qubit-Guard is a standalone, web-based enterprise security platform deployed within PNB's intranet DMZ. It is not
a component of any existing scanning tool — it is a purpose-built post-quantum cryptographic audit system. The product
performs outbound TLS handshake probing of public-facing assets and does not require installation of agents on target
servers, ensuring zero disruption to live banking services.
The system implements a T riad Scanning Architecture that simultane ously addresses all three cryptographic attack
surfaces in a modern bank: the public web portal (Pillar A), the remote-access VPN gateway (Pillar B), and the machine-
to-machine API layer (Pillar C). This unified approach fills a critical gap — no existing commercial scanner performs a
combined PQC-readiness assessment across all three pillars in a single orchestrated workflow .

Pillar B - VPN/TLS Engine - T argets standard VPN gateway ports (e.g., 1 194/UDP  for OpenVPN, 443/TCP  for standard SSL-VPNs). Evaluates TLS tunnel

Pillar C - API Security Engine & Dev/T est Integration - Scans REST , GraphQL, and SOAP  endpoints for mTLS configurations.   For application-layer token
analysis (JWT), the system includes an 'API Compliance Sandbox' where developers upload strictly non-production, dummy tokens or static OpenAPI
schema definitions to validate signing algorithms (alg: RS256, ES256, ML-DSA-65) before deployment.
